Who likes holding down CTRL?

BootstrapMultiSelect is a javascript plugin for Twitter Bootstrap 3 to help you create friendly multi-select components, created and maintained by Morris Singer.

Quick start

Clone the repo, git clone git://github.com/morrissinger/BootstrapMultiSelect.git, or download the latest release.

Add CSS, or SASS to your project, as appropriate, and add JS. Then, just be sure any select component you want to use this on has data-toggle=multiselect.
